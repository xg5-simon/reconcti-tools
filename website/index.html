Platform-Specific Setup

PowerShell
# Extract the archive Expand-Archive rdap-win-x64.zip -DestinationPath C:\Tools\rdap # Add to PATH (User level) $env:Path += ";C:\Tools\rdap" # Or add permanently [Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\Tools\rdap", "User") # Verify installation rdap --version
Bash
# Extract to /usr/local/bin sudo tar -xzf rdap-linux-x64.tar.gz -C /usr/local/bin/ # Make executable sudo chmod +x /usr/local/bin/rdap # Verify installation rdap --version
Bash/Zsh
# Extract to /usr/local/bin sudo tar -xzf rdap-osx-x64.tar.gz -C /usr/local/bin/ # Make executable sudo chmod +x /usr/local/bin/rdap # On macOS Catalina+, you may need to allow the binary sudo xattr -d com.apple.quarantine /usr/local/bin/rdap # Verify installation rdap --version

Quick Start

Once installed, you can start querying RDAP servers immediately:

Domain Lookup

Command
rdap domain example.com

Returns registration information including:

  • Domain status
  • Nameservers
  • Registrar information
  • Registration and expiration dates
  • DNSSEC details

IP Address Lookup

Command
rdap ip 8.8.8.8

Returns network allocation information including:

  • IP range and CIDR
  • Network name
  • Organization details
  • Country and region

ASN Lookup

Command
rdap asn 15169

Returns autonomous system information including:

  • AS name and description
  • Registration details
  • Organization information

Advanced Usage

JSON Output

Get raw JSON output for scripting and automation:

rdap domain example.com --json

Custom Server

Query a specific RDAP server directly:

rdap domain example.com --server https://rdap.verisign.com/com/v1/

Verbose Output

Enable detailed logging for debugging:

rdap domain example.com --verbose
